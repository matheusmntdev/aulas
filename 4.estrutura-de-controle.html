Exemplos de Estruturas de Controle em PHP

Aprenda a utilizar as estruturas de controle if, else e elseif em PHP com os exemplos a seguir.

1. Exemplo com if

O if é utilizado para verificar uma condição simples e, caso seja verdadeira, executar um bloco de código.



$numero = 5;

if ($numero > 0) {
    echo "O número é positivo.";
}

Explicação: Este código verifica se a variável $numero é maior que 0 e, caso seja, imprime a mensagem "O número é positivo."

2. Exemplo com if e else

O else é utilizado para fornecer um bloco de código alternativo, caso a condição do if seja falsa.



$idade = 17;

if ($idade >= 18) {
    echo "Você pode votar!";
} else {
    echo "Você ainda não pode votar.";
}

Explicação: Este código verifica se a variável $idade é maior ou igual a 18. Caso seja, a mensagem "Você pode votar!" será exibida, caso contrário, a mensagem "Você ainda não pode votar." será mostrada.

3. Exemplo com if, else e elseif

O elseif permite adicionar mais condições a serem verificadas, caso o if inicial não seja verdadeiro.



$idade = 30;

if ($idade < 12) {
    echo "Você é uma criança.";
} elseif ($idade >= 12 && $idade <= 17) {
    echo "Você é um adolescente.";
} else {
    echo "Você é um adulto.";
}

Explicação: Neste código, a condição $idade < 12 verifica se a pessoa é uma "criança". Se a idade estiver entre 12 e 17 anos, o elseif verifica se é um "adolescente". Caso contrário, o else classifica a pessoa como "adulto".

Explicação Adicional

  • if: Usado para verificar uma única condição.
  • else: Usado para fornecer uma alternativa quando a condição do if é falsa.
  • elseif: Permite adicionar condições adicionais a serem verificadas se a condição do if for falsa.

Essas estruturas são fundamentais para tomar decisões no seu código PHP, permitindo controlar o fluxo de execução de maneira eficiente.

Operador Ternário em PHP

O operador ternário é uma forma compacta de escrever uma expressão if-else em PHP. Sua sintaxe é a seguinte:


(condição) ? valor_se_verdadeiro : valor_se_falso;

Exemplo:


 
$idade = 20;
echo ($idade >= 18) ? "Você é maior de idade" : "Você é menor de idade";
