Estruturas de Repetição em PHP

As estruturas de repetição permitem executar um bloco de código várias vezes com base em uma condição. São úteis para automatizar tarefas repetitivas e manipular conjuntos de dados.

1. Estrutura for

A estrutura for é usada quando sabemos exatamente quantas vezes o loop será executado.

// Sintaxe:
for (inicialização; condição; incremento) {
    // Código a ser executado
}

Exemplo:

<?php
// Exibe números de 1 a 5
for ($i = 1; $i <= 5; $i++) {
    echo "Número: $i;" . PHP_EOL;
}
?>

2. Estrutura while

O while é usado quando não sabemos o número exato de iterações, mas temos uma condição a ser avaliada antes de cada execução.

// Sintaxe:
while (condição) {
    // Código a ser executado
}

Exemplo:

<?php
// Exibe números de 1 a 5
$i = 1;
while ($i <= 5) {
    echo "Número: $i;" . PHP_EOL;
    $i++;
}
?>

3. Estrutura do-while

A estrutura do-while garante que o bloco de código seja executado pelo menos uma vez, independentemente da condição.

// Sintaxe:
do {
    // Código a ser executado
} while (condição);

Exemplo:

<?php
// Exibe números de 1 a 5
$i = 1;
do {
    echo "Número: $i;" . PHP_EOL;
    $i++;
} while ($i <= 5);
?>

Controle de Loops

PHP permite controlar a execução de loops com as palavras-chave break e continue.

1. Uso de break

O break interrompe a execução do loop imediatamente.

<?php
// Para o loop ao encontrar o número 3
for ($i = 1; $i <= 5; $i++) {
    if ($i == 3) {
        break;
    }
    echo "Número: $i;" . PHP_EOL;
}
?>

2. Uso de continue

O continue interrompe a iteração atual e passa para a próxima.

<?php
// Pula o número 3
for ($i = 1; $i <= 5; $i++) {
    if ($i == 3) {
        continue;
    }
    echo "Número: $i;" . PHP_EOL;
}
?>
