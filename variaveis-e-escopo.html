Variáveis, Constantes e Escopo no PHP

1. Variáveis no PHP

Variáveis no PHP são espaços de armazenamento usados para guardar dados temporários. Elas podem armazenar diferentes tipos de valores, como números, strings, arrays, objetos, etc. A principal característica das variáveis é que seu valor pode ser alterado durante a execução do programa.

Declaração de variáveis:

  • Variáveis começam com o símbolo $, seguido pelo nome da variável.
  • As variáveis podem ser alteradas a qualquer momento.

Exemplo:

 
      $nome = "João";   // Variável do tipo string
      $idade = 25;      // Variável do tipo inteiro
      $altura = 1.75;   // Variável do tipo float

      echo "Nome: $nome, Idade: $idade, Altura: $altura";  // Exibe as variáveis

2. Constantes no PHP

Constantes são valores que não podem ser alterados durante a execução do programa. Ao contrário das variáveis, uma vez que uma constante é definida, ela mantém seu valor fixo e não pode ser modificada.

Declaração de constantes:

  • As constantes são declaradas usando a função define() ou a palavra-chave const.
  • O valor de uma constante não pode ser alterado após sua definição.

Exemplo com define():


 
      define("PI", 3.14159);
      echo PI;  // Exibe o valor de PI

Exemplo com const:


 
const TAXA_JURO = 0.05;
echo TAXA_JURO;  // Exibe o valor de TAXA_JURO

3. Escopo das Variáveis e Constantes

O escopo se refere à área do código onde uma variável ou constante pode ser acessada. Em PHP, existem diferentes tipos de escopo:

Escopo Global

Variáveis e constantes definidas fora de funções podem ser acessadas de qualquer parte do código.

Escopo Local

Variáveis definidas dentro de funções só podem ser acessadas dentro dessa função.

Exemplo de Escopo:


$x = 10; // Variável global

function testeEscopo() {
    global $x; // Acessando a variável global dentro da função
    echo $x;   // Exibe 10
}

testeEscopo();
echo $x; // Exibe 10, já que $x é global

Exemplo de Variável Local:


function exemploLocal() {
    $y = 20; // Variável local
    echo $y;  // Exibe 20
}

exemploLocal();
echo $y; // Gera erro, pois $y é local e não pode ser acessada fora da função
