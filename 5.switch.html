Estrutura switch em PHP

A estrutura switch permite realizar múltiplas comparações com uma variável de maneira mais organizada. Ao invés de usar vários if-else, o switch compara uma variável com diversos valores possíveis.

Exemplo:


$dia = 2;

switch ($dia) {
    case 1:
        echo "Domingo";
        break;
    case 2:
        echo "Segunda-feira";
        break;
    case 3:
        echo "Terça-feira";
        break;
    default:
        echo "Dia inválido";
}
